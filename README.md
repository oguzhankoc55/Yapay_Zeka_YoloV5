# Yapay_Zeka

## Python Bağımlılıklarını İndirmek İçin

Ana dizinde:
```
pip install -r yolov5/requirements.txt
```
## Web Server 'ı çalıştırmak için
Ana dizinde:
```
cd backend
npm i 
npm run start
```
